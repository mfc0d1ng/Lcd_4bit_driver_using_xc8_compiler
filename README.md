# LCD 4-bit driver
This is an implementation of lcd driver in 4-bit mode, it has been designed using xc8 compiler and provides 7 basic interfaces for controlling the lcd functionality and they are :- 
 * lcd_init();
 * lcd_cmd();
 * lcd_data();
 * lcd_cursor();
 * lcd_char();
 * lcd_str();
 * lcd_out();
